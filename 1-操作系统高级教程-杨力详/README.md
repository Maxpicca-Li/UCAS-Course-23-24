# 操作系统高级教程-杨力详

## 主要内容

基于 intel x86 IA32 指令集，会讲 Linux 源码，从基础需求推导OS的设计。

**后记：**大有裨益，强烈推荐每位CS同学来听。忠告是尽量课前看老师的书籍作为预习，前期跟着老师的代码思路。熟悉代码框架后，基本自己看教材就能学会了（老师不点名，dddd）。课堂上讲解的内容，大多都是教材的内容；但有时老师会引申部分专题，这个专题的探讨还挺有意思的，也可以多看群里讨论的专题。
